# 📊 Dashboard de Crímenes en Puerto Rico

Esta aplicación permite analizar y visualizar datos de crímenes reportados en Puerto Rico de manera interactiva.

---

## 🚀 Funcionalidades

- Filtrado dinámico por:
  - Área geográfica
  - Tipo de delito
  - Día de la semana
  - Periodo del día (AM / PM)

- Visualización de métricas clave:
  - Total de incidentes
  - Delito más frecuente
  - Área con más incidentes

- Gráficos interactivos:
  - Histograma de delitos
  - Mapa geográfico de incidentes

---

## 🗺️ Mapa interactivo

El mapa muestra:

- Ubicación de cada incidente
- Intensidad del delito según un índice de gravedad
- Ajuste automático del zoom según el área seleccionada

---

## 📈 Índice de gravedad de delitos

- Asesinato → 10
